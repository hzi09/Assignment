5. 각 범죄 별로 검거율을 계산하고, 각 검거율 데이터 column을 DataFrame에 추가해주세요.

   ```
6. 필요없는 column을 `del`을 사용하여 삭제해주세요.
    - 삭제해야할 column
    ```
    강간(검거),
    강도(검거),
    살인(검거),
    절도(검거),

    ```
7. DataFrame의 컬럼명을 `rename`을 사용하여 변경해주세요.

    ```
    '강간(발생)':'강간',
    '강도(발생)':'강도',
